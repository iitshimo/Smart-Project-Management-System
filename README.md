# Smart Project Management System

## Overview
A web-based project management system developed using ASP.NET Core MVC and SQL Server.

## Features
- User Authentication
- Project Management
- Task Management
- Role-Based Access Control
- Dashboard
- CRUD Operations

## Technologies
- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript

## How to Run

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Update the connection string in `appsettings.json`.
4. Apply database migrations.
5. Run the project.
